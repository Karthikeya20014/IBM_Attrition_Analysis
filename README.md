## ☕ IBM Employee Attrition Analysis Project

### Overview
This project analyzes Employees Attrition from a IBM Company 
and to extract insights and root causes for employees attrition 
and reccomendations to preserve the employees from getting attrited.

## 📊 Objectives
- Identify root causes for employee attrition.
- Analyze attrition by Gender, Department and more features
- Build KPI dashboard

- ## 📁 Project Structure
- dashboards/ → Excel dashboard
- sql/ → SQL queries
- reports/ → Final report

## 🛠 Tools Used
- MySQL
- Microsoft Excel
